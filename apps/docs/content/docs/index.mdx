## Getting Started

This guide will help you get started with BunNativeKit.

## Installation


```bash tab="bun"
bun add bnk
```

## Basic Usage

```typescript
import { jit } from "@bunnativekit/jit";

console.log("Basic JIT Add");

const result = await jit.zig(`
  export fn main(a: i32, b: i32) i32 {
    return a + b;
  }
`, 6, 7);

console.log(`6 + 7 = ${result}`);
console.log(`Type: ${typeof result}`);

```
